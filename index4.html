Design Principles for UI

Illustration by Harisankar Sahoo on Unsplash

What are they?

The 10 Usability Heuristics are essentially 'rules of thumb' when designing a website or app's user interface (UI).


These are meant to act as guidelines to help improve a webpage's UI to make it more user friendly - in other words, to provide a better user experience (UX).
